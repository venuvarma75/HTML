VISUAL STUDIO CODE

Visual Studio is an integrated development environment (IDE) developed by Microsoft. It is used to develop computer programs including websites, web apps, web services and mobile apps. Visual Studio uses Microsoft software development platforms including Windows API, Windows Forms, Windows Presentation Foundation (WPF), Microsoft Store and Microsoft Silverlight. It can produce both native code and managed code.

Features of VScode


Editing: Features like syntax highlighting, bracket matching, auto-indentation, and snippets help you be productive. You can also format your code, including setting indentation and moving curly braces.
Debugging: VS Code has an interactive debugger that lets you step through source code, inspect variables, and more. You can also use debugging actions like "Step Into" and "Step Out".
Version control: VS Code has built-in support for Git and other source control providers. You can work with source control without leaving the editor. Customization You can customize VS Code's UI and layout, including changing the theme, keyboard shortcuts, and preferences. You can also create profiles to quickly switch between different customizations.
User interface: VS Code has a minimap that gives you a high-level overview of your source code. You can also use Sticky Scroll to see the starting lines of currently visible nested scopes.
Extensions: You can install extensions to add functionality to VS Code.
Command line: VS Code has a command line interface (CLI) that lets you customize how the editor is launched.
Floating windows: You can open an editor in a floating window to move it to another place on your monitor.
