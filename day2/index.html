ELEMENTS OF HTML

There are three types of elements in html

1.BLOCK LEVEL ELEMENT

2.IN-LINE BLOCK ELEMENT

3.IN-LINE ELEMENTS

1.BLOCK LEVEL ELEMENTS:The tag which occupies space and creates a new line

EX:h1,h6,p,div,schematic elements

2.IN-LINE ELEMENT:The element which occupies only space and does not create new line and we can apply width,height,margin padding properties.

EX:img,button,iframe,input

3.IN-LINE ELEMENT:The element which occupies only space and cannot new line and cannot apply width,height,margin padding top and bottom and margin top and bottom.

EX:formatting tags,b,strong,i
